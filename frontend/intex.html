How It Works
STEP 01
Input Text
Paste a news article or headline. The more content provided, the more accurate the model's result.
STEP 02
TF-IDF Vectorisation
Your text is converted into numerical features using Term Frequency–Inverse Document Frequency weighting.
STEP 03
Naive Bayes Prediction
A trained Multinomial Naive Bayes classifier predicts Real or Fake with a confidence score.
About This Project
FINAL YEAR PROJECT · BSc IT
Fake News Detection System
Built with a Node.js + Express backend, a Python scikit-learn ML model, and a responsive HTML/CSS/JS frontend. The model uses TF-IDF vectorisation and Multinomial Naive Bayes classification. This tool is for educational purposes — always verify news with trusted sources.
